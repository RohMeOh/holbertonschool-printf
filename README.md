
# # _printf – Holberton School Project

This project is part of the Holberton School curriculum.

In this project, we built our own version of the C `printf` function, called `_printf`.  
It prints formatted output to the standard output, counts how many characters it prints, and returns that number.

Right now, our `_printf` handles:
- %c (characters)
- %s (strings)
- %% (printing a percent sign)
